
# 数学题生成器

一个简单的加减法题目生成器. 供打印使用(或口算).

一次最多生成50题, 正好一张A4纸.

![](./public/thumb.png)


## 功能

- [x] 10/20/50/100以内的计算
- [x] 加法
- [x] 减法
- [ ] 乘法
- [ ] 除法
- [x] 加法进位
- [x] 减法退位
- [x] 直接显示答案
- [x] 扫码显示答案
- [x] 打赏
- [x] 单题刷新
- [x] 题数控制
- [x] 题目是否可重复
- [x] 连加
- [x] 连减
- [x] 填空
- [ ] 混合运算

- 伪ts
